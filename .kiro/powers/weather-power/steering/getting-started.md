# Getting Started with Weather Power

This guide shows you how to use the Weather Power effectively.

## Quick Start

1. Activate the power to see available tools
2. Use `get_weather` for current conditions
3. Use `get_forecast` for upcoming weather

## Common Workflows

### Check Current Weather
Ask for weather in any city:
- "What's the weather in Seattle?"
- "Get current conditions for London, UK"
- "Temperature in Tokyo"

### Get Forecast
Request multi-day forecasts:
- "5-day forecast for Paris"
- "What's the weather going to be like in Sydney this week?"

### Unit Preferences
Specify your preferred units:
- Use "imperial" for Fahrenheit
- Use "metric" for Celsius (default)

## Tips

- Include country codes for accuracy (e.g., "Paris, FR" vs just "Paris")
- The power uses mock data for demonstration
- In production, connect to a real weather API service
